# IAM: Usuarios y Grupos

## ¿Qué es IAM?
IAM es un servicio global de AWS que permite gestionar usuarios y permisos.

## Buenas prácticas
- No usar el usuario root
- Usar usuarios IAM
- Asignar permisos mediante grupos

## Práctica realizada
- Creación de usuario IAM
- Creación de grupo Admin
- Asignación de permisos AdministratorAccess

## Aprendizajes
- IAM no depende de regiones
- Los grupos simplifican la administración
